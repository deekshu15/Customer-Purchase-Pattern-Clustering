# Customer Purchase Pattern Clustering

This repository contains code and analysis for customer segmentation using K-Means clustering on mall customer data. The project provides a terminal-based clustering script, supporting utilities, and an analysis notebook.

## What changed
- The original Streamlit UI was removed from active use and backed up to `app.py.bak`.
- The current runnable CLI entrypoint is available as `app.py.bak` (rename to `app.py` to restore the CLI).

## Project Summary
- Input: `data/Mall_Customers.csv` — customer records (ID, Gender, Age, Annual Income, Spending Score)
- Output: `data/clustered_customers.csv` — input with assigned `Cluster` column (generated by the CLI script)
- Core modules: `clustering.py` (clustering logic), `config.py` (settings), `utils.py` (visual helpers for the original dashboard)

## Quick Start (recommended)

1. Create and activate a virtual environment:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1   # Windows PowerShell
# or: source venv/bin/activate  # macOS / Linux
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the clustering script (use the venv python):

```powershell
& "${PWD}\venv\Scripts\python.exe" app.py.bak
```

This will print summary output (best k, silhouette scores, cluster sizes, profile means) and write `data/clustered_customers.csv`.

## Notebook

You can reproduce or extend the analysis interactively with the notebook:

```bash
jupyter notebook mall_customer_segmentation.ipynb
```
